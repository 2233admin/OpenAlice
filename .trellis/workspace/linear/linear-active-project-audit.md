## Scope

Audit active XAR/WEA/OMC Linear projects against `XartPro Linear Operating Standard v1`.

## Work

- List active/backlog projects.
- Classify each project as Class A / B / C.
- Check whether each Class C project has:
  - initiative
  - project status/lead
  - milestones/phases
  - parent program issue
  - child issues
  - dependency graph
  - handoff/QA issue
- Add missing structure or create follow-up issues.

## Acceptance

- Active complex projects no longer rely only on labels or loose issues.
- Each audited project has a visible Linear grouping surface.
- Findings are posted as a project update or issue comment.
