The goal is not to lock or migrate every project.

The goal is to avoid rebuilding the same Linear/project-management wheel for every future project.

Use OpenAlice as the reference implementation and `XartPro Project Reuse Kit v1` as the reusable kit:

- Class A: one issue is enough.
- Class B: parent + focused children.
- Class C: full OpenAlice-style spine, docs, milestones, dependencies, and handoff.

Do not retrofit unrelated projects unless they become active and the complexity actually needs this structure.
